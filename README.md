# S5-Client NodeJS SDK

An S5-Client in nodejs for Decentralized content-addressed storage network S5.

## Installing

1. Clone this repo.
2. cd s5-client-nodejs
3. npm install

## Documentation

For S5 documentation, please see [the S5 SDK docs](https://docs.s5.ninja/).
